# LiteXLoader - 方块 API

[<< 返回目录](README.md)

在LXL中，使用「方块对象」来操作和获取某一个方块的相关信息。

## 获取一个方块对象

通过注册**事件监听**函数，或者调用某些**返回方块对象**的函数，来获取到BDS给出的方块对象  

<br>


## 方块对象 - 属性

每一个方块对象都包含一些固定的对象属性。对于某个特定的方块对象`bl`，有以下这些属性

| 属性    | 含义         | 类型     |
| ------- | ------------ | -------- |
| bl.name | 方块名称     | `String` |
| bl.pos  | 方块所在坐标 | `IntPos` |

这些对象属性都是只读的，无法被修改
